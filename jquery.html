Les fonctions de base

  • Nous devons inclure la balise <script> dans le document HTML avant de créer un lien vers un fichier JavaScript qui utilise la bibliothèque jQuery.
  • .ready() attend que le DOM soit entièrement chargé, et invoque alors la fonction de rappel spécifiée.
  • Nous pouvons cibler un élément soit par sa classe, son id ou son genre:
    1. Cibler avec la classe: $('.menu-button');
    2. Cibler avec l'id: $('#menu-button');
    3. Cibler avec son genre: $('p');
  • Nous pouvons également stocker un élément dans une variable, nous nommerons alors cette variable en commençant par un $, ce n'est pas obligatoire mais c'est une bonne pratique. Ex: const $menuButton = $('.menu-button');
  • La méthode on() ajoute des gestionnaires d'événements aux objets jQuery. La méthode prend deux paramètres : une chaîne de caractères déclarant l'événement à écouter (le gestionnaire) et une fonction de rappel à lancer lorsque l'événement est détecté.

Les effets

  • La méthode hide() et show() font apparaître ou disparaître un élément. La méthode toggle() fait les deux en même temps et agit comme un interrupteur on/off. Cette méthode peut prendre un paramètre: la vitesse de l'effet(slow, fast, ms)
  • La méthode fade est utilisée pour faire apparaître ou disparaître un élément avec un fondu
    1. fadeIn() est utilisée pour faire apparaître en fondu un élément caché.
    2. fadeOut() est utilisée pour faire disparaître en fondu un élément caché.
    3. fadeToggle() combine fadeIn et fadeOut.
    4. fadeTo() permet de réaliser un fondu vers une opacité donnée (valeur comprise entre 0 et 1). Sytaxe: $(selector).fadeTo(speed,opacity,callback);
  • Les méthodes jQuery slide font glisser les éléments vers le haut et vers le bas.
    1. slideDown() sert à faire apparaître un élément vers le bas.
    2. slideUp() sert à faire disparaître un élément vers le haut.
    3. slideToggle() combine slideDown() et slideIn().

Parcourir le DOM

  • .children() nous permet de sélectionner les enfants d'un élément. Cette méthode sélectionne uniquement les enfants direct.
  • .parent() nous permet de sélectionner le parent d'un élément.
  • .siblings() nous permet de sélectionner les éléments qui se trouvent au même niveau dans le DOM que l'élément sélectionné.
  • .closest() nous permet de sélectionner l'élément le plus proche avec le sélecteur spécifié en paramètre.
  • .next() et .prev() nous permettent de sélectionner l'élément sibling suivant ou précédant.
  • .find() nous permet de sélectionner un élément singulier qui est le descendant d'un autre élément. Il retournera tous les descendant qui répondent au critère.
