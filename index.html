Git et GitHub

Les commandes de base

  • git init crée un nouveau dépôt Git.
  • git status inspecte le contenu du répertoire de travail et de la zone de transit.
  • git add filename ajoute des fichiers du répertoire de travail à la zone de transit.
  • git add -A ajoute tous les fichiers du répertoire de travail et sous-dossiers à la zone de transit.
  • git add . ajoute tous les fichiers du répertoire de travail à la zone de transit.
  • git diff montre la différence entre le répertoire de travail et la zone de transit.
  • git commit stocke définitivement les changements de fichiers de la zone dans la base de données locale.
  • git log montre une liste de tous les commits précédents.

Comment revenir sur ses pas?

  • HEAD représente le commit le plus récent.
  • git show HEAD montre le commit le plus récent?
  • git checkout HEAD filename supprime les changements dans le répertoire de travail.
  • git reset HEAD filename annule les changements de fichiers dans la zone de transit.
  • git reset commit_SHA permet de revenir à un commit précédent dans votre historique de commit.
